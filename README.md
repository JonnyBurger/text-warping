Demo of the capabilities of the [`warpPath()`](https://www.remotion.dev/docs/paths/warp-path) API in Remotion

## Commands

**Install Dependencies**

```console
npm i
```

**Start Preview**

```console
npm start
```

**Render video**

```console
npm run build
```

**Upgrade Remotion**

```console
npm run upgrade
```
